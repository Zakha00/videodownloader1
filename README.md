# 🤖 Video Downloader Bot

Высоконагруженный Telegram-бот для скачивания видео с монетизацией через подписку на канал.

## 📁 Структура

```
bot/
├── main.py          — точка входа, все хендлеры
├── database.py      — работа с SQLite (aiosqlite)
├── downloader.py    — скачивание через yt-dlp
├── keyboards.py     — inline-кнопки
├── requirements.txt
└── .env.example
```

## ⚡ Быстрый старт

### 1. Переменные окружения (Render → Environment)

| Переменная | Описание |
|---|---|
| `BOT_TOKEN` | Токен от @BotFather |
| `CHANNEL_ID` | `@username` или `-100xxxxxxxxxx` |
| `CHANNEL_URL` | `https://t.me/your_channel` |
| `CHANNEL_NAME` | Название канала |
| `ADMIN_IDS` | Твой Telegram ID (через запятую) |

### 2. Настройка канала

1. Создай канал в Telegram
2. **Добавь бота в канал администратором** (обязательно! иначе проверка не работает)
3. Права бота в канале: достаточно "Просмотр участников"
4. Узнай свой ID: напиши боту @userinfobot

### 3. Деплой на Render

```
Build Command:  pip install -r requirements.txt
Start Command:  python main.py
```

## 🚀 Масштабирование под 100k пользователей

### Сейчас (Render Free / Starter):
- SQLite справится до ~5k активных пользователей/день
- Узкое место — CPU при параллельных скачиваниях

### Когда вырастешь:

**Шаг 1 — Render Starter ($7/мес):**
- 512MB → 2GB RAM
- 0.5 CPU
- Хватит до ~20k пользователей/день

**Шаг 2 — Очередь задач:**
```python
# Заменить run_in_executor на Celery + Redis
# Это даст параллельную обработку скачиваний
```

**Шаг 3 — PostgreSQL вместо SQLite:**
```
DATABASE_URL=postgresql://...
```
SQLite не поддерживает много параллельных записей. При 10k+ DAU переходи на PostgreSQL (Render даёт бесплатно 1GB).

**Шаг 4 — Несколько воркеров:**
```python
# gunicorn / несколько процессов
# или aiogram с несколькими Bot инстансами
```

## 💰 Монетизация ($300–500/мес)

### Текущая схема (подписка на канал):
- 5 бесплатных → подписка на канал
- Монетизируй канал через рекламу (@telega_in, @sociate)
- 10k подписчиков канала = $100–300/мес от рекламы

### Дополнительные источники дохода:

**1. Telegram Stars (встроенная оплата):**
```python
# После 5 бесплатных — 50 Stars за безлимитный месяц
# Telegram берёт 30%, тебе 70%
```

**2. Реклама в caption видео:**
```python
AD_CAPTION = "📥 Скачано\n💎 @your_premium_channel — эксклюзив!"
```

**3. Premium-подписка через Stars:**
- 100 Stars = 30 дней безлимита
- При 1000 платящих = ~$500/мес

## 📊 Антиспам и защита

- ✅ Rate limit: 1 запрос в 10 секунд на пользователя
- ✅ Лимит размера файла: 50 МБ
- ✅ Логирование ошибок в БД
- ✅ Обработка всех исключений yt-dlp
- ✅ Удаление файлов сразу после отправки

## ⚠️ Важно

- Файлы в Telegram ограничены **2 ГБ**, но бот ограничен **50 МБ** (иначе Render упадёт)
- Видео удаляются сразу после отправки — диск не забивается
- База данных `bot.db` — добавь её в `.gitignore`
